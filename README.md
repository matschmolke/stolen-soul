## Stolen Soul
2D Roguelike / Dungeon Crawler game created using the Unity engine, in which the player takes on the role of a hero who has been stripped of their memories by an evil necromancer. The goal of the game is to progress through successive levels, recover the lost equipment, and defeat the necromancer.
